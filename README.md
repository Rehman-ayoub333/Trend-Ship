# TRENDSHIP 2025
## LX Hausys Design Trend Platform

A high-fidelity, interactive web experience showcasing the 2025 interior design trend: **SY(E)NERGY**.

### Features
- **9 Immersive Pages**: Home, Design Trend, Themes, Exhibition, Collection, Report, Lookbook, About, and Contact.
- **Advanced Motion**: GSAP-driven scroll animations, Three.js 3D organisms, and smooth Lenis scrolling.
- **Responsive Design**: Fluid typography and layouts optimized for all devices.
- **Functional Components**: Material filtering, modal systems, and dynamic navigation overlays.

### Tech Stack
- **GSAP 3.12.2**: For timeline and scroll-based motion.
- **Three.js r128**: For 3D generative visuals.
- **Lenis 1.0.19**: For smooth viewport scrolling.
- **Standard HTML5/CSS3**: Self-contained pages with inline styles for portability.

### Setup
Simply run a local server in the root directory:
```bash
# Using npm
npx serve .

# Or using Python
python -m http.server 5000
```
Visit `http://localhost:5000` to view the site.
